About This Project

This Netflix Recommendation Engine demonstrates multiple machine learning approaches to collaborative filtering and matrix completion.

üßÆ Algorithms Implemented

Expectation-Maximization (EM) Clustering

Approach: Uses Gaussian Mixture Models to cluster users by their rating patterns.

How it works: Groups users into clusters, then predicts missing ratings using cluster-specific means.

Best for: Discovering user archetypes and handling diverse preferences.

K-Nearest Neighbors (KNN)

User-based: Finds users with similar rating patterns and averages their ratings.

Item-based: Finds movies with similar rating patterns and recommends accordingly.

Best for: Interpretable recommendations based on similarity.

Matrix Factorization

Approach: Decomposes the user-movie rating matrix into latent factors.

Methods: Non-negative Matrix Factorization (NMF) and Singular Value Decomposition (SVD).

Best for: Capturing complex patterns and achieving high accuracy.
